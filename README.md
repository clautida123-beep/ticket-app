# 🚀 AI-Driven Helpdesk - Astra Techn

Système de gestion de tickets IT intelligent conçu pour optimiser le support technique en entreprise.

## 👥 À propos du projet
Développé par le groupe **Astra Techn** dans le cadre de notre projet de fin d'année (L2 RTM & IA). 
L'application automatise la classification des tickets via IA et gère les flux réseau (SMTP/IMAP) pour une réponse en temps réel.

## 🛠 Architecture Technique
* **Frontend :** Interface utilisateur réactive (React/Vue.js).
* **Backend :** API RESTful (PHP/XAMPP).
* **Intelligence Artificielle :** Modèle de classification NLP (Python/Scikit-Learn).
* **Infrastructure :** Serveur local Apache (XAMPP), Base de données MariaDB.

## 📋 Répartition des rôles
- **Chef de Projet :** Gestion et coordination globale.
- **Frontend :** UI/UX et intégration.
- **Backend :** API, DB et logique métier.
- **IA / Data :** Entraînement du modèle et traitement NLP.
- **Réseau / DevOps :** Intégration RTM, CI/CD et déploiement local.
- **QA & Doc :** Tests unitaires et manuel utilisateur.
